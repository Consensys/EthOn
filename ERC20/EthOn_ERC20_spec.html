EthOn ERC20 - An ERC20 Extension to EthOn

This ontology extends EthOn with concepts that can be used to describe ERC20 compliant Tokens.

Allowance

A Token Allowance Provided by a Token Balance. The Delegate Account of the Allowance can spend a fixed amount of Tokens on behalf of the token owner.

Status unstable
URI http://ethon.consensys.net/ERC20/Allowance
Superclasses EthOnERC20Concept
Properties include allowance hasSpender
Used with providesAllowance

Superclasses TokenEvent

DelegateTokenTransfer

A Delgate Token Transfer is a Token Transfer executed by a Spender that has been given an Allowance to spend.

TokenTransfer

A Token Transfer is the record of an amount of tokens being transferred from one Account to another Account. If the Token complies to ERC-20 the Transfer is logged by an Event.

Status unstable
URI http://ethon.consensys.net/ERC20/TokenTransfer
Superclasses EthOnERC20Concept
Subclasses DelegateTokenTransfer OwnerTokenTransfer
Properties include value from to
Used with hasTransfer

TransferEvent

A Transfer Event is emitted when a Token Transfer from one Account to another Account takes place.
